## Description

Assign all readonly members.

## Motivation

In the following code the readonly member B is never assigned.

```c#
public class Foo
{
    public Foo(int a)
    {
        this.A = a;
    }

    public int A { get; }

    public int B { get; }
}
```

## How to fix violations

Assign a value to B in constructor or in initializer (showing borth in the example below):

```c#
public class Foo
{
    public Foo(int a)
    {
        this.A = a;
        this.B = 2;
    }

    public int A { get; }

    public int B { get; } = 3;
}
```
